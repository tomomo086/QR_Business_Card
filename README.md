# 📱 QRコード名刺プロジェクト

## 🎯 概要
3Dプリンター(Neptune 4 Pro)でQRコード付き名刺を作成。
GitHubポートフォリオへの直接アクセスを可能にする面談用インパクトツール。

## 📐 仕様
- **サイズ**: 60mm × 60mm × 2.0mm
- **QR部分**: 50mm × 50mm
- **材料**: 白PLA(ベース) + 黒PLA(QR)
- **URL**: https://github.com/tomomo086

## 📂 フォルダ構成
```
├── 3d_models/      # 3Dモデルファイル
├── docs/           # 設計資料・進捗
├── print_settings/ # 印刷設定・G-code
└── qr_assets/      # QRコード素材
```

## 🚀 実装手順
1. ✅ QRコード生成 (完了)
2. 🔄 3Dモデリング (進行中)
3. ⏳ マルチカラー印刷設定
4. ⏳ 面談準備
